# Marble Catch

A small browser-based marble catching game made with HTML, CSS, and JavaScript.

The player moves a cup left and right to catch falling marbles.
A simple console display is included to show internal values such as position, velocity, gravity, and rotation angle.

## Demo

https://tikuro69.github.io/marble-catch/

## Features

- Simple marble falling physics
- Keyboard control
- Catch / miss scoring
- Debug-style console display
- Rotating obstacle behavior

## Controls

- Left Arrow: move cup left
- Right Arrow: move cup right
- Space: restart the game

## Purpose

This is a small experimental game project.

The goal was to test simple motion, collision behavior, and visual feedback in a browser game.
The console display was added to make it easier to observe how the values change while the game is running.

## Tech Stack

- HTML
- CSS
- JavaScript
